Setup

This project has been tested running PHP 5.6.13 for Windows. It is started by navigating to the root and using the command php -S localhost:8000. The folder root is treated as the document root. The site can then be found by going to localhost:8000 in the browser.

It is also running on a CentOS 7 server being served by Apache. The url for this version is http://php.landonia.me/SimplePHPUpload

Operation

Home Page

The index page will allow users to choose a file (image) to upload and submit. It will also pick up a URL parameter with fileName= and try to find an image in upload by that name to display.
This home page also features a slider at the bottom which displays all of the uploaded images in all of their uploady glory. It checks for updates every 5 seconds and rebuilds if necessary.

Processing Page

This is the page the form submits back to. It will run some validation on the file (size/extension) and display error messages with a link back to the form if necessary. If there are no issues, the javascript will pick up and redirect back to the home page with the url parameter ?fileName= and the name of the uploaded file. This allows the home page to display the uploaded image back to the user.
